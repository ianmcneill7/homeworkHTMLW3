CSS = Cascading Style Sheets
Inline: use the style inside HTML Elements.
Internal: use a style inside the header.
External: using a link element to link to an external css file. (most common)
Inline css is h1 style="color:blue;"

External CSS, you just add a link to a CSS style sheet file in a directory with:
